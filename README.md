# Real_Estate_Price_Prediction
# 🏠 Bengaluru House Price Prediction

## Project Overview
This project is a **Machine Learning-based web application** that predicts residential property prices in Bengaluru using features such as location, total area (square feet), number of bedrooms (BHK), and bathrooms. The project covers the complete machine learning pipeline, from data preprocessing and model training to deployment through a Flask web application.

## Dataset Used
**Name:** Bengaluru House Price Dataset

**Source:** Kaggle – Bengaluru House Data

**Dataset Includes:**
- Property Location
- Total Square Feet
- Number of Bedrooms (BHK)
- Number of Bathrooms
- Price
- Other property attributes

## What I Did
- Cleaned and preprocessed **13,000+ housing records** by handling missing values, removing outliers, and transforming categorical features.
- Performed feature engineering and one-hot encoding to improve model performance.
- Trained and evaluated multiple regression models including **Linear Regression, Lasso Regression, and Decision Tree Regression** using **GridSearchCV**.
- Selected the best-performing **Linear Regression** model based on evaluation metrics and cross-validation.
- Developed a **Flask REST API** to serve real-time house price predictions.
- Built a responsive frontend using **HTML, CSS, and JavaScript** for user interaction and live predictions.
